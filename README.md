## КАК ЗАПУСКАТЬ ТЕСТЫ
### 1. Установка зависимостей
напишите в терминале следующую команду:
```
pip install -r requirements.txt
```
### 2. Запуск тестов
Все тесты находятся в папке `test/`. Для запуска, в терминале напишите:
```
python -m pytest
```
для более подробного вывода напишите:
```
python -m pytest -v
```
